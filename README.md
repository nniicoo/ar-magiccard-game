# ar-magiccard-game
1.项目简介
《AR 异界召唤：猜拳对战》是一款基于 Unity 与 Vuforia 引擎开发的增强现实（AR）互动小游戏，融合经典“石头剪刀布”玩法与虚拟角色召唤机制，打造沉浸式人机对战体验。玩家通过手机扫描实体召唤图卡，即可在现实空间中生成炫酷的魔法角色，并通过简洁的触控操作完成猜拳对战。胜利角色将释放专属技能动画与粒子特效，失败角色则触发动作表现，增强视觉冲击与互动反馈。游戏设有角色图鉴系统，支持角色解锁与信息展示，提升收集与重复游玩价值。项目适合用于虚拟现实课程教学、互动展示或移动端休闲娱乐，具有开发难度适中、创意十足、演示效果直观的特点。支持 Android 端运行，适配主流手机设备，体验轻量却精彩的 AR 竞技玩法，兼具技术实践性与展示效果，适合作为AR开发课程的示范案例。
2.交互与展示
1.图卡识别交互（AR图像识别）
交互方式：通过 Vuforia 引擎识别实体图卡
交互类型：现实世界图卡 + 虚拟角色绑定
展示效果：在图卡上方生成3D角色与召唤特效，实现“召唤”场景的沉浸式再现
2.玩家操作交互（触控式选择）
交互方式：
1.	开始界面：三个按钮，选择，开始，退出，声音开关
2.	选择界面：有介绍说明界面，有多种卡牌选择以及详情界面，选择后，图像识别就会生成玩家选择的角色，每个界面都有相应的返回按钮。
3.	游戏界面：界面弹出三个虚拟按钮【石头】【剪刀】【布】，以及玩家所选择的角色，还有角色的血量，以及返回按钮，特效按钮，用户进行石头剪刀布对战，总共五局。
3.胜负判定与反馈交互
选择完后会出现以下反馈：
1.	冻结三秒按钮
2.	显示选择结果文本，并进行倒计时
3.	释放对应的粒子特效
4.	展示人物对应的动作
5.	血量也会相应变化
倒计时结束后玩家重新选择，总共五局，平局不算，血量最多的胜利，玩家胜利就播放胜利歌进入胜利界面，失败就进入失败界面，然后可以选择重新开始游戏。
交互方式：系统自动比对玩家双方输入，判定胜负。类型：被动反馈式交互
展示内容：
胜者角色播放粒子特效（火焰、雷电等）+ 攻击动作，失败者播放闪避动作
同时弹出胜负提示文本（如：“You win!”）和双方的选择,以及倒计时，并冻结选项。
4.结果界面交互
交互方式：胜负结算后，用户通过按钮选择继续或退出
按钮设置：
[again] → 重启游戏流程
[exit] →关闭应用
展示反馈：弹窗UI浮现
